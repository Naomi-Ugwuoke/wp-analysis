Issues With the Webpage

Issue 1

The webpage seems to have a grid-style background running throughout the webpage (refer to image below). However, it is cut-off by sections with white backgrounds. As a result of this, there seems to be no continuity in the webpage.

Issue 2

Another issue with this website is the responsiveness of the webpages in relation screen size. On bigger screens, the episodes in the main section of the website have two columns, and on smaller screens, it has only one column. The problem is that depending on the size of the screen, the body of the website is displayed towards the left of the screen, leaving much whitespace on the right side. This observation can be seen in the image below:

Issue 3

At the beginning of the webpage, there is a header section. However, the header section does not start the page. There is blank space above the header. This diverts the user's attention from on the focus point of the website. The focus should instead be on the header bars available. Also, on the navigation menu provided, there is a link to a “STORE”, however, there is also another link that says “SHOP”. These two navigation items point to the same location. There is no need to duplicate the same destination twice in the same section. A suggestion that can be made for this section is to move the social links to the footer instead of the header, and to remove the store link. The search button could also be moved to the main section of the webpage.
